# Product Compare App

Мобільний застосунок для порівняння товарів на React Native з використанням Expo. Дозволяє користувачам переглядати каталог товарів різних категорій та порівнювати їх характеристики у вигляді інтерактивної таблиці з автоматичним виділенням кращих параметрів.

## Можливості

- **Каталог товарів** - переглядання товарів різних категорій (Накопителі, Одяг, Кросівки, Бижутерія)
- **Порівняння товарів** - вибір кількох товарів та порівняння їх характеристик
- **Візуальне виділення** - автоматичне підсвічування кращих параметрів зеленим кольором
- **Навігація** - легка навігація між екранами через React Navigation
- **State Management** - управління станом через Context API
- **React Native** - нативний код для Android

## Швидкий старт

### Вимоги

- Node.js (версія 16 або вище)
- npm або yarn
- Expo CLI

### Встановлення та запуск

1. **Клонування репозиторію**
```bash
git clone https://github.com/FosterLand/RIK_Zalik.git
cd RIK_Zalik
```

2. **Встановлення залежностей**
```bash
npm install
```

3. **Запуск застосунку**
```bash
npm start
```

4. **Відкриття в Expo Go**
   - Натисніть:
     - `a` для Android емулятора
     - `w` для веб-браузера

## 💡 Основні компоненти

### ProductCard
Відображає інформацію про окремий товар:
- Зображення, назву, описання
- Ціну та рейтинг
- Кнопку для додавання/видалення зі порівняння

### CompareTable
Інтерактивна таблиця для порівняння характеристик:
- Динамічне видобування характеристик
- Автоматичне визначення найкращих значень
- Обробка різних типів даних (текст, числа, одиниці виміру)
- Виділення кращих значень зеленим фоном

### SelectionContext
Управління станом вибраних товарів:
- `addToSelection()` - додавання товару
- `removeFromSelection()` - видалення товару
- `clearSelection()` - очищення списку
- `isSelected()` - перевірка статусу вибору

## 📖 Як користуватися

1. **Перегляд каталогу**: На головному екрані ви бачите список усіх доступних товарів
2. **Додавання до порівняння**: Натисніть кнопку на карточці товару
3. **Перегляд таблиці**: Перейдіть на екран "Порівняння" для перегляду таблиці
4. **Аналіз**: Порівняйте характеристики - кращі параметри виділені зеленим кольором
5. **Видалення**: Натисніть на товар у таблиці для видалення зі порівняння
